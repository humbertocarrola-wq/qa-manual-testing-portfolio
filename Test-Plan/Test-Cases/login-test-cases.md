# Login Test Cases

## TC-001 - Valid Login

Steps:
1. Go to login page
2. Enter valid username
3. Enter valid password
4. Click login

Expected Result:
User is redirected to dashboard.

---

## TC-002 - Invalid Password

Steps:
1. Go to login page
2. Enter valid username
3. Enter invalid password
4. Click login

Expected Result:
Error message is displayed.
